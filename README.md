LeafStrength es una aplicación web progresiva (PWA) para planificar, seguir y optimizar entrenamientos de fuerza y musculación.

## Características

- 📊 **Gestión de Ejercicios**: Crea y administra tu biblioteca de ejercicios personalizados
- 📅 **Mesociclos**: Diseña planes de entrenamiento con diferentes modos (Full Body, PPL, Weider, etc.)
- 🔄 **Rotación Inteligente**: Distribuye automáticamente los ejercicios en varias sesiones
- 📈 **Seguimiento de Progreso**: Registra series, pesos, repeticiones y RPE
- 🏆 **Records Personales**: Visualiza tus mejores marcas automáticamente
- 📱 **PWA**: Funciona offline y se puede instalar en dispositivos móviles
- 💾 **Exportación/Importación**: Respalda y restaura tus datos fácilmente

## Modos de División Disponibles

- **Full Body**: Todo el cuerpo en cada sesión
- **Torso/Pierna**: Alternancia entre torso y piernas
- **PPL (Push/Pull/Legs)**: Clásico tirón/empuje/pierna
- **PPL+Brazos**: PPL con día dedicado a brazos
- **Weider 5/6 días**: División clásica de culturismo
- **Personalizado**: Selección manual de ejercicios por sesión

## Instalación

1. Clona el repositorio
2. Abre `index.html` en tu navegador
3. ¡Listo! Puedes usarla localmente o subirla a cualquier hosting estático

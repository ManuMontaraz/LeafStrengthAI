# Política de Seguridad de LeafStrength

## Reportar Vulnerabilidades

Si descubres una vulnerabilidad de seguridad en LeafStrength, por favor:

1. **NO abras un issue público** en GitHub
2. Envía un email a: contacto@manumontaraz.es
3. Incluye detalles suficientes para reproducir el problema
4. Espera nuestra respuesta antes de divulgar públicamente

## Qué Reportar

- Vulnerabilidades XSS
- Problemas de LocalStorage
- Fugas de datos
- Problemas de CSP
- Vulnerabilidades en dependencias

## Qué NO Reportar

- Problemas de rendimiento
- Sugerencias de características
- Bugs funcionales (usa issues públicos para estos)

## Compromiso

Me comprometo a:
- Responder en menos de 48 horas
- Mantener confidencialidad sobre el reportero
- Reconocer contribuciones válidas (si el reportero lo desea)
- Solucionar problemas críticos en menos de 30 días

## Ámbito

Esta política aplica a:
- La aplicación web en leafstrength.app
- Los archivos JavaScript y CSS
- Las APIs (si existieran)
- La configuración de servidor

Gracias por ayudar a mantener LeafStrength seguro.
